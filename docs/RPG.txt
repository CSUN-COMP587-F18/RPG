	int stat

  int maxHealth

  int defense

Class Player extends Creature
  int level
	int experience
	Weapon weapon
	Shield shield
	Helm helm
	Body body

Class Monster extends Creature

Dungeon level determines stats by randomly assigning the number of stat points that would be available at that level randomly to attack, defense, and health.
